#### ✨ Task 1.2: Stage 4-B TF-IDF + 감성분석 (3~4일)

**목표**: 키워드 매칭 → TF-IDF 자동 추출 + 감성점수 추가

```python
# Step 1: TF-IDF
from sklearn.feature_extraction.text import TfidfVectorizer
from textblob import TextBlob

tfidf = TfidfVectorizer(
    max_features=100,
    ngram_range=(1, 2),
    min_df=5,
    max_df=0.8
)
X_tfidf = tfidf.fit_transform(reviews_target['review_text'])
feature_names = np.array(tfidf.get_feature_names_out())

# 상위 20개 키워드
top_idx = X_tfidf.toarray().sum(axis=0).argsort()[-20:]
top_keywords = feature_names[top_idx]
print(f"Top keywords: {top_keywords}")

# Step 2: 감성분석
reviews_target['sentiment'] = reviews_target['review_text'].apply(
    lambda x: TextBlob(x).sentiment.polarity
)
reviews_target['subjectivity'] = reviews_target['review_text'].apply(
    lambda x: TextBlob(x).sentiment.subjectivity
)

# Step 3: 월별 추이
monthly_sentiment = reviews_target.groupby('year_month').agg({
    'sentiment': 'mean',
    'subjectivity': 'mean',
    'review_text': 'count'  # 리뷰 수
})
monthly_sentiment.plot()
```

**기대 결과**:
- TF-IDF: 상위 20개 자동 추출 키워드
- 감성: 월별 sentiment 평균 (2016년 10월 극저값 확인)

## 💬 FAQ

**Q. 왜 권태기 모델 AUC가 0.65로 낮은가?**

A. 권태기 단계는 "서서히 자연 이탈"하는 경향이 있어, 급격한 행동 변화(스킵률 급증 등)가 적습니다. 따라서 단순 행동 신호(ML)만으로는 포착이 어렵고, 정성적 신호(리뷰의 불만도)와 함께 봐야 합니다. NLP 감성분석으로 보완할 예정입니다.

**Q. "버그가 2016년 10월 이탈의 원인"이라는 가설을 어떻게 입증할 것인가?**

A. 시계열 상관 분석을 합니다:
1. ML: 2016-08 vs 2016-10 스킵률 비교 (상승 증명)
2. NLP: 리뷰에서 "버그", "추천", "스킵" 키워드 빈도 추이
3. 교차: 둘의 시간 동기화 여부 + 상관계수 계산

**Q. A/B 테스트는 언제부터 시작되는가?**

A. Stage 6 액션 플랜이 완성된 후, 7월 말~8월 중순부터. 그 전에 "샘플 크기 계산", "기간 설정", "평가 메트릭" 정의를 마쳐야 합니다.
