### 1. 프로젝트 소개

### 1-1. 주제 선정 배경

음악 스트리밍 서비스는 구독 기반 비즈니스 모델이다. 이 모델에서는 신규 고객을 유치하는 것만큼이나 기존 고객을 유지하는 것이 수익성에 직결된다. 일반적으로 신규 고객 획득 비용은 기존 고객 유지 비용의 5~7배에 달한다고 알려져 있으며, 따라서 이탈 가능성이 높은 고객을 미리 파악하고 선제적으로 대응하는 것이 비즈니스 관점에서 훨씬 경제적이다.

기존의 이탈 예측 연구들은 대부분 "이탈할 것인가, 아닌가"를 맞히는 분류 모델 성능에 집중한다. 그러나 예측만으로는 실제로 이탈을 막기 위해 무엇을 해야 하는지 알 수 없다. 따라서, 목표는 "어떤 고객 불만이 → 어떤 행동 변화로 이어지고 → 결국 이탈까지 연결되는가"라는 인과적 흐름을 데이터로 규명하는 것이다. 단순 예측에 그치는 것이 아니라, 이탈의 원인을 밝혀내는 것이 최종 목표이다.

### 1-2. 이탈 기준 정의

구독 만료일로부터 30일 이내에 유효한 신규 구독이 없으면 이탈로 간주하며, 취소 후 30일 이내 재구독 시에는 이탈로 보지 않는다. KKBox의 구독 모델이 대부분 30일 단위 월정액이기 때문에, 매월 재구독하는 패턴이 일반적임을 반영한 기준이다.

### 4. 선행연구 조사

| 논문/연구 | 핵심 방법론 | 우리 프로젝트 참고점 |
| --- | --- | --- |
| Bryan Gregory (WSDM 1위, 2018) | 시계열 피처 엔지니어링 + 단일 XGBoost | 정적 feature보다 행동 변화 추세(temporal feature)가 훨씬 예측력 높음. "복잡한 모델보다 좋은 피처 엔지니어링이 더 중요하다" 입증 |
| An Ensemble Approach | LightGBM + XGBoost + LR 2단계 앙상블, 3축 피처 | 회원/결제/로그 각 축별로 시간 구간별 피처를 분리 구성하는 방식 참고 |
| GAT 네트워크 기반 | 유저를 노드로 연결한 그래프 학습 | 결제 패턴·취소 이력이 청취 시간보다 이탈 영향 더 큼. 클래스 불균형 처리(churn 가중치 10) 필요 |
| XAI 기법 적용 연구 | SHAP, LIME, PDP/ICE 기법 | 부스팅 계열 모델 + XAI로 이탈 요인 수치화. 유저 세그먼트별 맞춤 리텐션 전략 도출 가능 |
| 생존 분석 연구 | Cox, DeepHit, Right Censoring | 이탈 여부뿐 아니라 "언제 이탈하는가(타이밍)"까지 예측. 현재 active user ≠ non-churn(아직 이탈 이벤트 미발생)이라는 관점 |
